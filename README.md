# Marty Bonacci Portfolio

Portfolio website showcasing 40+ years of coding experience across product development and AI collaboration frameworks.

**Live site:** [martybonacci.dev](https://martybonacci.dev)

## Projects Featured

- **CustomCult** - Algorithm-driven custom snowboard design platform
- **SpecSwarm** - Claude Code plugin for spec-driven development
- **Frame Injection** - AI collaboration research and soft skills transfer
- **Four Minds Pattern** - Cognitive architecture for AI orchestration

## Tech Stack

- React Router v7 (Framework Mode)
- TypeScript
- Tailwind CSS v4
- Vercel
